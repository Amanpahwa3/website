Permission-less vs Permissioned network end-points:

A network end-point is permissioned if it only wants to accept connections from a known set of peers (trusted_peers) identified by their network identity keys. This does not mean that the other end-point of a connection also needs to run in permissioned mode -- a network end-point running in permissioned mode will connect to or accept connections from an end-point running in permissionless mode as long as the latter is in its trusted peers set.
