This trait defines interfaces to be implemented by a storage write client.

There is a 1-1 mappings between each interface provided here and a LibraDB API. A method call on this relays the query to the storage backend behind the scene which calls the corresponding LibraDB API. Both synchronized and asynchronized versions of the APIs are provided.
